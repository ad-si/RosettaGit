'''DM''' (short for '''Dream Maker''') is a programming language built for the [BYOND game engine.](http://www.byond.com/)

The official language guide can be found [here](http://www.byond.com/docs/guide/), and the language reference can be found [here](http://www.byond.com/docs/ref/index.html).
