''nawk'' is another name for ''The One True Awk'', the original implementation by Alfred Aho, Peter Weinberger and Brian Kernighan.

''nawk'' is "New Awk", and refers to the [the new features from 1985](https://en.wikibooks.org/wiki/An_Awk_Primer/Nawk). Versions from before 1985 would be ''oawk'' for "Old Awk". There is no reason to use ''oawk'', but some systems had both, and the [Heirloom Toolchest](https://heirloom.sourceforge.net/tools.html) still includes both ''oawk'' and ''nawk''.

Because of the release of [source code of The One True Awk](http://www.cs.princeton.edu/~bwk/btl.mirror/), some [BSD](https://rosettacode.org/wiki/BSD) systems now include ''nawk'' as <tt>/usr/bin/awk</tt>.
