Visual C++ is [Microsoft](https://rosettacode.org/wiki/Microsoft)'s C++ implementation, part of Visual Studio. After version 6 it was replaced with Visual C++.Net (see [Wikipedia's Visual C++ page](https://en.wikipedia.org/wiki/Visual_C++) for more info).
