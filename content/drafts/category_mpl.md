MPL is a multi-precision (long integer) arithmetic library for AutoHotkey.

Discussion and usage can be found on its [topic page](https://www.autohotkey.com/board/topic/19940-long-integer-arithmetic-library/), or the latest version can be [directly downloaded](https://www.autohotkey.com/boards/code_downloader.php?id=11949&part=2). It currently only works in 32bit versions of AHK.
