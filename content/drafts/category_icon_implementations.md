* [[:Category:Unicon|Unicon the Unified Extended Dialect of Icon]]

The following are other variants of Icon
* [ProIcon](https://www.cs.arizona.edu/icon/ftp/doc/ipd276.pdf) - a version of Icon for the Macintosh]
* [Seque](https://www.cs.arizona.edu/icon/ftp/doc/tr85_2.pdf) - an Icon variant with streams. Streams are a data object that produces and retains values.
* [Rebus](https://www.cs.arizona.edu/icon/ftp/doc/tr84_9.pdf) - a SNOBOL4 / Icon hybrid
* MTIcon - a multi-threading implementation of Icon, superseded by Unicon
* IDOL - a pre-processor extension to Icon to facilitate object oriented code, superseded by Unicon
