[range-v3](https://github.com/ericniebler/range-v3) is a an implementation of a formal proposal to add range support to the C++ standard library. It is an evolution of the popular [boost.range](http://www.boost.org/doc/libs/release/libs/range/doc/html/index.html) library (range-v2).
